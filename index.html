Html5 loader jQuery plug-in

(Beta version 0.2)

This plug-in creates a nice preloader useing html5 canvas API to draw the percentage during the page loading. Now it works just with images but I hope soon it will work also with audio and video objects.

Demos

  • Nice glow graphic + automatic page preloding
  • FUN! Draw more than one canvas without any reason
  • Automatic preload of an html page + Circular default preloader
  • Line preloader it's only one of a lots future features
  • Passing data with JSON file
