Html5 loader jQuery plug-in

(Beta version 1.0)

Now you can preload your HTML 5 page with this nice plug in, passing data via JSON. You can customize it changing colors and style. It can preload video, audio, images and js.

Demos

  • COOL (font famyly + glow + lineWidth) customization
  • Glow circular preloader
  • FUN! Draw more than one canvas without any reason
  • Circular default preloader
  • Line preloader
